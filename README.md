Project Overview:
This repository contains a Power BI Sales & Revenue Analysis Dashboard developed as part of the Tata Data Visualisation: Empowering Business with Effective Insights virtual experience program offered by Tata Group via Forage.

The project focuses on transforming raw business data into meaningful insights to support CEO- and CMO-level decision-making using effective data visualisation techniques.

Business Objectives=>

The dashboard was designed to answer key leadership questions, including:

How is revenue trending over time?
Which countries contribute the most to revenue and sales quantity?
Who are the top revenue-generating customers?
How can leadership quickly identify high-performing regions and customers?
